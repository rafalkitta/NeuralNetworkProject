  • multiplyMatrixes(a:b:)

    Normal mtrixes multiplication - matrix product. If a is an n × m matrix and b is an m × p matrix, their matrix product ab is an n × p matrix, in which the m entries across a row of a are multiplied with the m entries down a columns of b and summed to produce an entry of ab.

    Note

    Parameters order does matter.

    Swift

    b

    Second matrix to multiplication

    Return Value

    Result of given matrixes multiplication

  • multiplyMatrixes2(a:b:)

    Matrix multiplication like * in Python. Each value from a matrix is multiplyed by value in exactly the same position in b matrix.
